== Description ==

> This plugin requires a <a href="http://www.sendinblue.com" rel="nofollow">SendinBlue</a> account.

###Integrate SendinBlue with Gravity Forms
If you use <strong>SendinBlue</strong> email service and the Gravity Forms plugin, you're going to want this plugin!

Integrate your Gravity Forms forms so that when users submit a form entry, the entries get added to SendinBlue. Link any field type with SendinBlue, including custom fields!

== Installation ==
1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer.
2. Activate the plugin
3. Go to the plugin settings page (under Forms > Settings > SendinBlue)
4. Enter the information requested by the plugin.
5. Click Save Settings.
6. If the settings are correct, it will say so.
7. Follow on-screen instructions for integrating with SendinBlue.
